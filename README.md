# Astrbot Lofter解析插件

## ✨ 简介
一个为Astrbot设计的插件，支持解析Lofter分享链接，并将文本与图片内容转发过来

仅需要配置Lofter的Cookie，无需第三方API

目前仅在QQ中进行了测试，暂不支持视频内容

## ⚙️ 配置选项
| 配置项          | 描述           | 默认值    | 说明          |
| --------------- | ----------------- | --------- | ---------|
| `lofter_cookie` | Lofter(乐乎)cookie | "" | 可以用浏览器获取 |
